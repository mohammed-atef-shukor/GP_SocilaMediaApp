# 📱 Social Media Platform (Frontend)

A modern social media web application built using Angular, focusing on clean UI, performance, and scalable architecture.

> ⚠️ Note: This repository contains **frontend only**.  
> Backend is built separately using Spring Boot.

---

## 🚀 Features

- 🔐 Authentication UI (Login / Sign Up)
- 📝 Create, edit, and delete posts
- ❤️ Like / Unlike posts
- 🧑 User profiles UI
- ⏱️ Human-readable timestamps (e.g. "2 hours ago")
- 🔄 HTTP Interceptors for API handling
- ⚡ Reactive state management using Angular Signals
- 📱 Responsive design

---

## 🛠️ Tech Stack

### Frontend
- Angular 19
- TypeScript
- RxJS
- Angular Signals
- HTML5 / SCSS

### Backend (External)
- Spring Boot (REST API)
- JWT Authentication
- MySQL / PostgreSQL

---

## 🧠 Architecture Highlights

- Clean component-based architecture
- Reusable UI components
- Separation of concerns (services, models, components)
- Centralized API handling with interceptors
- Scalable folder structure
